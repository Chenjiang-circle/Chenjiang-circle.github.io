圈圈 | 2019年度总结

发表于 2020-01-16 | 分类于 年度总结

圈圈成长记 | 2019个人年度总结

序言

光阴似箭，日月如梭，这句话真的没错。随着自己的年龄的增长，体重却一直居低不上，感觉时间过得越来越快，以前都是论“天”过，现在一不小心就发现一周已经过去了。转眼间大二的学习生活已经过去一半，在这紧张的期末备考阶段，我心（bu）血（xiang）来（fu）潮（xi），想总结一下自己的这一年成长、变化。

第一次写年度总结，没有经验，于是参照了掘金、CSDN等平台上的年度总结的结构。

WHO AM I

我觉得有必要先介绍一下我自己。

option content
姓名 圈圈
性别 男
年龄 19岁（马上就是奔三的人了）
学校 河南某“农业”双一流大学，在读本科二年级学生
专业 数据科学与大数据技术专业
爱好 踢足球、撸猫
单身与否 圈圈心中已经住着一个也只会住着一个小可爱了，你说我还单身吗？
性格 特别容易相处的一男的。但是内心藏着一只大家平常见不到的圈圈。

关于我为什么叫圈圈，以后有时间了，我将从盘古开天辟地给大家讲起，让大家清楚又明白地知道为啥我叫圈圈。

Flag

近期的目标非常明确，保研。之前的我，还一直在犹豫，到底是读研还是直接工作。就在写这篇总结之前，我下定决心要读研，而且要保研！与自己的导师见面的时候，我说出了自己的打算，但是表示自己有点担心，因为保研名额有限，竞争压力较大，害怕自己得不到保研名额。但是导师的一句话让我不再有所顾虑，无所畏惧地选择去争保研名额！

她是这样说的：“成事在人。”

确实，机会都是自己争取的，我也并不差，为什么不争取一把呢？“梦想”总是要有的，万一实现了呢？

专业知识


作为一名计算机专业的学生以及未来的程序员，我深知不断学习的重要性。有必要在这里总结一下自己在专业知识方面一年的收获和积累。

2019上半年

2019年上半年，我还是大一，自己在学院“实验室”自学Java。我的Java入门书籍是《Head First Java》，确实这本书对我这种小白来说确实非常之友好，大量的图片（甚至书中出现了毫无厘头的顺口溜），让我骄傲于自己一天50页的阅读速度之外，也学会了Java的基本语法。不知不觉中，Java殿堂的大门已经被我这个懵懂的“IT界的小孩”打开了。看完《Head First Java》之后，我对Java基础语法的整体框架有了一个比较全面的认识，然后我选择去看了一本人人推荐的《Thinking in Java》，这本书对我还说确实是劝退级别的书——砖头一样的厚度，蚂蚁一样的小字。虽然内容很枯燥，但是我从中学到了什么是面向对象，对面向对象有了初步的认识。在整个过程中我还不断地做笔记，坚持每天把自己的笔记推到GitHub上面，但是呢，《Java编程思想》最终没能坚持看完……

暑假是我进步最快、学东西最多、收获最大、成就满满的一段时间。暑假前一个月，我没有回家，也没有出去打工，选择了在学校学习。因为大一上学期的时候，我学过 HTML、CSS 和 JavaScript 的基础语法，也做了几个网页练了练手，所以在大一下学期学了几个月的java基础语法之后，选择在暑假尝试去学习Java Web。学习资料是“实验室”学长给的 Java Web 视频，是培训机构的那种付费视频。这种视频讲得快，很多原理也不讲，但是实用性很强，在边看视频边敲代码两周之后，Java Web项目的整个基本流程就搞清楚了。

看完视频感觉自己会了，但是心里没有底呀！于是乎就和同学组个小团队，一起从 0 开发了一个音乐论坛。大一上学期的时候，我们一起学了git的基本用法，所以，我们选择用GitHub + git进行团队开发。这是第一次做“小项目”，完全没有经验，没有写开发文档，前后端没有确定接口，几个人讨论了一下音乐论坛该有什么样的功能之后，红红火火恍恍惚惚地就开始了。可想而知，我们之后会遇到多少问题。那段时间，晚上10点手指离开键盘，洗漱完之后，躺在床上想怎么解决今天遇到的 bug，想还有什么地方可以完善，想为什么我这么菜，想我什么时候可以变强，想自己未来的生活……然后进入梦乡。不过早上不到6点钟就醒了，想想这个音乐论坛还有好多功能没有实现，就完全没有了睡意，翻身起床，就又坐在了电脑旁边，一坐就是一天……如此生活循环了小半个月。

最终我们的音乐论坛在7月末的时候初步完成了，回过头来看这一个月（看视频 + 做音乐论坛），只能用一句话来形容：值！我知道，我所掌握的Java知识也好，Java Web只是也好，只能算刚入门，要学得东西还有好多。在开发论坛的时候，我就已经遇到了许多“培训视频”中不曾提及的问题和知识，不过没有关系，靠着自己大一上学期练就的搜索能力，和自认为还可以的学习能力，顺利解决了一个又一个难题。暑假学习完美收官。

2019年下半年

下半年我也成了大二学长，专业课也变得多了起来，开设了Linux入门、python基础、Java高级程序设计等。回顾整个下学期，发现自己并没有学多少新的东西（除了专业课），课余时间比较少，自己想学的东西又可多，就变得有些迷茫，有些无所适从。看了几十页的《Java核心技术卷一》就没再继续往下看，又去看了《图解HTTP》，但是也没有看完，在此期间我用GitHub Page搭建了静态博客，之后又去看网课学数据结构，因为好多教材也好、视频也好，都是用c和c++进行演示的，所以只学了c++基本语法的我，又回过头来看了一段时间的《C++ Primer Plus》，数据结构才刚刚学完线性表，就快到了期末，所有的课外学习计划就一律暂停，一切以期末考试为重。

期末考试结束，不想早早地的回家被老妈嫌弃于是选择留在学校学点东西。因为自己报了下学期的“蓝桥杯”python组，所以决定在学校刷一刷竞赛题。我选择了在PAT的竞赛题库中做练习，刷了10道题左右之后，就对比较熟练地掌握了python的基础语法，因为python比较慢，所以也迫使我不断地对算法进行优化，使代码运行时间变短。

生活


目前还是一名大学生，没有收入来源，所以更谈不上月薪、年薪和跳槽。每天的生活就是吃饭、睡觉和学习，偶尔会娱乐一下，不过后者占的比例非常的小。自己的身体素质不算好，但是也不算差，因为肠胃不强大，所以一直非常的瘦。其他的方面也没什么好谈的了，跟其他大学生的生活大差不差。

但是！但是！但是！

我在今年下半年，有女朋友啦！！

快给我羡慕！羡慕！我都羡慕我自己呢！

不用多说啥，不想撒宝贵的粮食，总之一句话：爱她、爱生活、爱代码。
